# Advanced Computing Lab Practicals

This repository contains laboratory practicals, assignments, datasets, and Jupyter notebooks for the **Advanced Computing for Biological Systems** course.

## Course Overview

The course introduces computational and machine learning techniques used in biological and biomedical data analysis. Practical sessions cover statistical analysis, data visualization, regression, hypothesis testing, classification, neural networks, and pattern recognition using Python and Jupyter Notebook.

## Repository Structure

```text
├── Lab_1   : Statistical visualization and exploratory data analysis
├── Lab_2   : Measures of central tendency and descriptive statistics
├── Lab_3   : Data analysis using Python
├── Lab_4   : Data preprocessing and visualization
├── Lab_5   : Linear Regression
├── Lab_6   : Logistic Regression
├── Lab_7   : Hypothesis Testing
├── Lab_8   : Support Vector Machines (SVM)
├── Lab_9   : Machine Learning Applications
├── Lab_10  : Artificial Neural Networks (ANN)
├── Lab_11  : Convolutional Neural Networks (CNN) and Applications
```

## Software Requirements

* Python 3.10+
* Jupyter Notebook / JupyterLab
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn
* PyTorch

Install dependencies using:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn torch jupyter
```

## Usage

Clone the repository:

```bash
git clone https://github.com/dasprabir/Advanced_Computing_Lab_Practicals.git
cd Advanced_Computing_Lab_Practicals
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```

## Learning Outcomes

Upon completion of these practicals, students will be able to:

* Analyze biological and biomedical datasets.
* Perform statistical analysis and hypothesis testing.
* Build predictive machine learning models.
* Apply classification and regression algorithms.
* Develop neural-network-based solutions for real-world problems.
* Visualize and interpret experimental data.
